Welcome to Read Mode Extension!

Thank you for installing the Read Mode Extension for Chrome! This extension is designed to enhance your reading experience by providing features to easily manage and customize the way you read web content. Whether you're reading articles, e-books, or any other form of text, the Read Mode Extension makes it easier to focus on the content.

Features

  • Reading Mode: Toggle reading mode to simplify web pages for easier reading. The extension removes distractions and adjusts the layout for a better reading experience.
  • Highlight & Notes: Highlight important text and add notes to it. Save and view your highlights and notes later.
  • Font Size Adjustment: Customize the font size of the text to suit your reading preference using an easy-to-use slider.
  • Dark Mode: Switch to dark mode to reduce eye strain during night-time reading.
  • Read Aloud: Listen to the content being read aloud with adjustable speech settings. Perfect for multitasking or accessibility needs.
